# Construct an `npi_results` S3 object

Creates an `npi_results` S3 object from a tibble. See
[`validate_npi_results`](validate_npi_results.md) for other requirements
for this class.

## Usage

``` r
new_npi_results(x, ...)
```

## Arguments

- x:

  A tibble

## Value

A tibble with S3 class `npi_results`
